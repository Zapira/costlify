# COSTLIFY

**Solusi Cerdas untuk Mengelola Biaya dan Keuntungan Bisnis UMKM**

COSTLIFY adalah aplikasi gratis untuk UMKM yang membantu menghitung biaya produksi (HPP), menentukan harga jual, dan memantau keuntungan secara otomatis tanpa perlu ilmu akuntansi.

---

## ğŸ¯ Fitur Utama

- **Perhitungan HPP Otomatis** - Hitung Harga Pokok Produksi dengan mudah dan akurat
- **Penetapan Harga Jual** - Tentukan harga jual yang tepat berdasarkan biaya dan margin keuntungan
- **Pemantauan Keuntungan** - Pantau profit secara real-time tanpa perhitungan manual
- **Tanpa Ilmu Akuntansi** - Antarmuka sederhana yang mudah dipahami siapa saja
- **Gratis** - Aplikasi sepenuhnya gratis untuk semua pelaku UMKM
